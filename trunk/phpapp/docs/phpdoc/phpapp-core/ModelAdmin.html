		setFieldGroups (line 878)
	
 
	
	

Configures form field grouping and display options for  field groups.


Example of the format of the $groups parameter:  
  1.  array(
    2. 

  2.      array('name' => null,
    3. 

  3.         'fields' => array('title', 'content', 'type')),
    4. 

  4.      array('name' => 'Form Action',
    5. 

  5.         'fields' => array('action'),
    6. 

  6.         'displayConditions' => array('type' => 'FORM')),
    7. 

  7.      array('name' => 'SEO Fields',
    8. 

  8.         'fields' => array('meta_keywords', 'meta_description'),
    9. 

  9.         'collapse' => true),
    10. 

  10.     )
    11. 

Group options:  
  • name: The group heading, or null for no heading.
  • fields: An array of field names that belong to
		the group.
  • collapse: Displays the field group as a single line
 		initially, that can be expanded with a click.
  • displayConditions: Defines rules that determine when
		this field group should be displayed.  The value of this
		option is a field => value array.  When the current
		form value for field is equal to value, the
		field will be displayed; otherwise it will be hidden.  This
		behavior is dynamic; fields will be shown and hidden via
		Javascript as form values change.
