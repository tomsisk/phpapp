Class AbstractCursorIterator

Description
Description | Descendents | Methods (details)

Implements interfaces:

  • ArrayAccess (internal interface)
  • Iterator (internal interface)
  • Countable (internal interface)

An iterable class that wraps a database result, functioning as a scrollable cursor.

AbstractCursorIterator can be used in foreach loops or with direct array index access.
