Class ManyToOneField

Description
Description | Descendents | Vars | Methods (details)

Defines a many-to-one relation between two models.

In SQL terminology, this is a FOREIGN KEY field.

Example:

  1.  class Book extends Model {
  2.      public function configure() {
  3.          $this->id = new IntegerField('ID', array('pk' => true));
  4.          $this->author = new OneToManyField('Author', 'Author');
  5.      }
  6.  }
  7.  
  8.  class Author extends Model {
  9.      public function configure() {
  10.          $this->id = new IntegerField('ID', array('pk' => true));
  11.          $this->name = new CharField('Name', 100);
  12.      }
  13.  }
  14.  
  15.  // Author object is not loaded yet; we only made a single query
  16.  $book = $bq->one();
  17.  // Author is loaded as soon as we call the ->author field,
  18.  // and we can access its fields like normal
  19.  $authorname = $book->author->name;

Because relation fields are lazy-loading, you avoid excessive queries while retaining the benefit of transparent access to the related objects in your code.
