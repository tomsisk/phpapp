Class TextField

Description
Description | Descendents | Vars | Methods (details)

Field contains textual data.

>TextField has several custom option:</p> blank: If false, blank values are not allowed.
case: Specifies that the input text should be transformed before saving to the data store. Possible values: "upper" (convert to uppercase), "lower" (convert to lowercase), "capitalize" (capitalize the first letter), "title" (capitalize the first letter of every word)
