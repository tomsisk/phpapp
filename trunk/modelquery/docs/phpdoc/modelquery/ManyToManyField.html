Class ManyToManyField

Description
Description | Vars (details) | Methods (details)

A field that returns a set of models related to this one, using a third intermediary Model object to create a many-to-many mapping.

This field has several custom options:

joinField: The name of the field on the intermediary model that links to the source model's primary key. Defaults to the source model's name, lower-cased. targetField: The name of the field on the intermediary model that links to the target model's primary key. Defaults to the target model's name, lower-cased. Example:

  3.          $this->id = new IntegerField('ID', array('pk' => true));
  4.          $this->genres = new ManyToManyField('Genres', 'Genre', 'AuthorGenre'):
  5.      }

  17.          $this->name = new CharField('Name', 100);

Although you need the intermediary field definition to manage the relationships, you should usually not need to access it directly, unless you need to store additional information about the relation. ModelQuery adds and removes records of that model as needed to maintain the relationship links.
